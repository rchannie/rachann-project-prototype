🔄 Alur Sistem Utama

1. Klien Membuat Pesanan
Klien upload brief, tentukan harga estimasi, max revisi, dan deadline
2. Pembayaran DP
Klien bayar DP (biasanya 50%), admin verifikasi pembayaran
3. Admin Setup Project
Admin assign designer, buat milestone template, project status = active
4. Designer Upload Desain
Designer upload file untuk setiap milestone, status berubah ke "Review"
5. Admin Send Preview
Admin review desain, lalu kirim notifikasi preview ke klien
6. Klien Review & Approve
Klien bisa approve atau request revisi dengan catatan

📱 Fitur Mobile Apps (Client & Designer)
